Data Science Projects - Dicoding Indonesia

projectwebsite.example.com

I completed these two projects as a graduation requirement for the Data Science certification by Dicoding Indonesia. In the data analysis course, I used Streamlit to create a bike rental dashboard by showing information about the total rentals per month, the highest and lowest number of rentals based on time, and the number of rentals based on time and air temperature.

The second project, in the machine learning course, required me to perform customer segmentation. For this project, I used several data preprocessing techniques such as handling missing values, removing duplicate data, normalizing features, detecting and handling outliers, encoding and binning, and analyzing the data using visualization (Distortion Score Elbow, Centroid), as well as K-Prototypes for clustering. Then, I continued with classification using three machine learning models, namely Support Vector Machine (SVM), K-Nearest Neighbors (KNN), and Naive Bayes. The highest result was achieved by Naive Bayes, with accuracy, precision, recall, and F1-Score all reaching 0.997.
