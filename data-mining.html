Data Mining Projects - NYCU

projectwebsite.example.com

In my opinion, this project has quite a heavy weight as the work I did to complete the Data Mining course in fall 2025, offered by the Institute of Data Science and Engineering, National Yang Ming Chiao Tung University, because it was one of the most challenging courses for me. There were a total of 3 assignments and 1 final project, and all of them used the Kaggle platform, so I had to compete with all students in the class to get the highest rank.

In the first assignment, I was only allowed to use NumPy, so I did a lot of data preprocessing such as replacing invalid values, melting and pivoting the dataset, checking autocorrelation, handling outliers using Robust Scaler, and others. Then I combined a sliding window algorithm and gradient descent from scratch to perform linear regression. In the second assignment, the topic was unsupervised anomaly detection using the MVTec AD dataset (unlabeled, no class) without using any pre-trained model. Therefore, I decided to use the KMeans algorithm for clustering and a Convolutional AutoEncoder. The third assignment focused on news recommendation, which I handled using several preprocessing techniques, Sentence Transformers (DistilBERT), and a multilayer perceptron. Finally, for the final project, the given problem was about knowledge tracing prediction, which I solved using MiniLM and XGBoost.

Assignments

  • PM2.5 Prediction (Linear Regression) using NumPy
  • Anomaly Detection with MVTec AD Dataset
  • News Recommendation Prediction

Final Project

  • Knowledge Tracing Prediction
